# Session 1 (25 Minutes)

## Revision: Logical Operators (`&&`, `||`, `!`)

### Learn

### AND (`&&`)

Returns true **only if both conditions are true**.

Example:

```c
if(age >= 18 && age <= 60)
```

This means the age must be **between 18 and 60**.

---

### OR (`||`)

Returns true if **at least one condition is true**.

Example:

```c
if(marks < 0 || marks > 100)
```

This checks whether the marks are **invalid**.

---

### NOT (`!`)

Reverses the result.

Example:

```c
if(!(age >= 18))
```

Equivalent to:

```c
if(age < 18)
```

# Session 2 (45 Minutes)

## Topic: `while` Loop

### Syntax

```c
while(condition)
{
    // statements
}
```

### Flow

1. Check the condition.
2. If true, execute the loop body.
3. Repeat until the condition becomes false.

### Example

Print numbers from 1 to 5.

```c
#include <stdio.h>

int main() {
    int i = 1;

    while(i <= 5) {
        printf("%d ", i);
        i++;
    }

    return 0;
}
```

**Output:**

```text
1 2 3 4 5
```
