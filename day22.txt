# Session 1 (45 Minutes)

## Topic: `do-while` Loop

### What is a `do-while` Loop?

A `do-while` loop executes the loop body **at least once**, even if the condition is false.

### Syntax

```c
do
{
    // statements
} while(condition);
```

### Flow

```
Execute Body
      ↓
Check Condition
      ↓
True → Repeat
False → Exit
```

### Example

```c
#include <stdio.h>

int main()
{
    int i = 1;

    do
    {
        printf("%d ", i);
        i++;
    } while(i <= 5);

    return 0;
}
```

**Output:**

```
1 2 3 4 5
```

## Session 2 (35 Minutes)

## Difference Between `for`, `while`, and `do-while`

| Feature                         | for | while | do-while |
| ------------------------------- | --- | ----- | -------- |
| Condition checked before loop   | ✅   | ✅     | ❌        |
| Executes at least once          | ❌   | ❌     | ✅        |
| Best for known iterations       | ✅   | ⚠️    | ❌        |
| Best for menus/input validation | ❌   | ⚠️    | ✅        |

### Example

```c
int i = 10;

while(i < 5)
{
    printf("%d", i);
}
```

Output:

```
Nothing
```

Now:

```c
int i = 10;

do
{
    printf("%d", i);
} while(i < 5);
```

Output:

```
10
```
