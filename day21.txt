# Session 1 (45 Minutes)

## Topic: `for` Loop

### What is a `for` Loop?

A `for` loop is used when you know **how many times** a loop should execute.

### Syntax

```c
for(initialization; condition; update)
{
    // statements
}
```

### Example

Print numbers from 1 to 10.

```c
#include <stdio.h>

int main()
{
    int i;

    for(i = 1; i <= 10; i++)
    {
        printf("%d ", i);
    }

    return 0;
}
```

### Flow

```
Initialization
      ↓
 Check Condition
      ↓
 Execute Body
      ↓
    Update
      ↓
Back to Condition
```

# Session 2 (35 Minutes)

## `while` vs `for`

| while                            | for                            |
| -------------------------------- | ------------------------------ |
| Initialization outside           | Initialization inside          |
| Update inside loop               | Update in header               |
| Best when iterations are unknown | Best when iterations are known |

### Example

### while

```c
int i = 1;

while(i <= 5)
{
    printf("%d ", i);
    i++;
}
```

### for

```c
for(int i = 1; i <= 5; i++)
{
    printf("%d ", i);
}
```
