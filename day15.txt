# Day 5 Tasks

**Total Time:** ~2 Hours 30 Minutes

## Session 1 (50 Minutes)

### Topic: Conditional Statements (`if`, `if-else`)

Study:

* `if`
* `if-else`
* Comparison using relational operators

Example:

```c
#include <stdio.h>

int main() {
    int age;
    printf("Enter age: ");
    scanf("%d", &age);

    if (age >= 18) {
        printf("Eligible to vote");
    } else {
        printf("Not eligible to vote");
    }

    return 0;
}
```

Understand:

* Condition
* True block
* False block
