# Section B – Output Prediction (15 Marks)

### Q6

```c
#include <stdio.h>

int main()
{
    int i = 1;

    while(i <= 5)
    {
        printf("%d ", i);
        i += 2;
    }

    return 0;
}
```

Predict the output.

### Q7

```c
#include <stdio.h>

int main()
{
    int i;

        printf("%d ", i);
    }

    return 0;
}
```

Predict the output.
